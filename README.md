# 💳 Credit Card Fraud Detection Using Machine Learning

## 📌 Project Overview

This project focuses on detecting fraudulent credit card transactions using Machine Learning algorithms. Credit card fraud detection is an important real-world application of Artificial Intelligence and Machine Learning in the banking and finance sector.

The project uses multiple classification algorithms to identify fraudulent transactions from a highly imbalanced dataset.

---

# 🎯 Objectives

- Analyze credit card transaction data
- Detect fraudulent transactions using Machine Learning
- Compare different classification algorithms
- Evaluate model performance using accuracy and classification metrics
- Identify the best-performing model

---

# 📂 Dataset

Dataset used in this project:

🔗 Kaggle Dataset:
